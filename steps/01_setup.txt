Flask is a lightweight Python web framework, often referred to as a microframework.


to use flask we need to install Flask framework
        pip install flask

now import Flask in 1st app
        from flask import Flask
create a app of flask
        app = Flask(__name__)
now you have to route each request from this app using
        @app.route
        def hello():
            return "Hello world"

        app.run(debug=True)

now you can run this code using python 

            debug=True used to keep updating elements in our code and donot need to run again and again
